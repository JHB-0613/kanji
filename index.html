데이터 분석 개념 퀴즈 5탄 (코드 제외)

각 선지를 **클릭**하면 해당 문항의 정답과 해설이 바로 표시됩니다. (클릭 후 재선택 불가)

1. 특정한 분석 모형을 활용해 데이터로부터 결과값이나 시각화를 도출하는 단계이며, 주로 정량적인 방법을 사용해 의미 있는 정보를 추출하는 활동은 무엇인가요?
  • a. 데이터 수집 (Data Collection)
  • b. 데이터 분석 (Data Analysis)
  • c. 데이터 해석 (Interpretation)
  • d. 시각화 (Visualization)
정답: b. 데이터 분석 (Data Analysis)
데이터 분석은 정량적인 방법을 사용해 정보 추출 및 결과값 도출을 수행하는 단계입니다.
2. 다음 중 One-way ANOVA 분산분석을 사용하여 평균 차이의 유의성을 검증하기 위한 조건으로 가장 적절한 것은?
  • a. 단일 집단이 알려진 특정 평균값과 일치하는지 검증할 때
  • b. 동일 대상의 조건 전후 변화(두 시점)를 검증할 때
  • c. 하나의 범주 변수에 따른 세 개 이상의 집단 간에 유의미한 차이가 있는지 검증할 때
  • d. 두 변수 간의 선형성이 일치하는 정도를 측정할 때
정답: c. 하나의 범주 변수에 따른 세 개 이상의 집단 간에 유의미한 차이가 있는지 검증할 때
ANOVA는 두 집단 이상의 평균 차이를 검증할 때 사용되며, One-way는 하나의 범주 변수를 기준으로 합니다.
3. 데이터 전처리(Data Cleansing) 과정에서 수행하는 주요 작업이 아닌 것은?
  • a. 분석 결과를 바탕으로 가설을 검증하고 결론을 내린다.
  • b. 결측치(Missing Value)를 처리하거나 대체한다.
  • c. 중복 데이터를 관리한다.
  • d. 이상치(Outlier)를 제거하거나 보정한다.
정답: a. 분석 결과를 바탕으로 가설을 검증하고 결론을 내린다.
가설 검증 및 결론 도출은 **데이터 해석(Interpretation)** 단계에서 수행됩니다.
4. 수치형 데이터 중, 측정 가능한 범위 내에서 **임의의 실수** 값을 가지며 직접 연산이 가능한 자료형은 무엇인가요?
  • a. 이산형 (Discrete)
  • b. 명목형 (Nominal)
  • c. 서열형 (Ordinal)
  • d. 연속형 (Continuous)
정답: d. 연속형 (Continuous)
연속형은 특정 범위 내의 임의의 실수 값을 가지며, 이산형은 정수 값을 가집니다.
5. 데이터를 시각화하여 비교할 때, **평균과 표준편차**를 사용하여 데이터의 중심 위치와 흩어진 정도를 파악하는 데 유용한 방법은 무엇인가요?
  • a. 정규 분포 (Normal Distribution) 곡선
  • b. Boxplot (상자 그림)
  • c. 산점도와 추세선
  • d. 이동평균선
정답: a. 정규 분포 (Normal Distribution) 곡선
정규 분포 곡선은 평균을 중심으로 종 모양을 이루며, 표준편차에 따라 폭이 달라져 분포를 비교하기 쉽습니다. Boxplot은 중앙값과 사분위수를 사용합니다.
6. 미리 정의된 스키마/구조를 가지며 스프레드시트나 관계형 $\text{DB}$에 저장되는 데이터와, $\text{XML, JSON}$처럼 태그나 마커로 유연한 구조를 가지는 데이터를 각각 무엇이라고 하나요?
  • a. 비정형 데이터 / 정형 데이터
  • b. 서열형 데이터 / 명목형 데이터
  • c. 연속형 데이터 / 이산형 데이터
  • d. 정형 데이터 / 반정형 데이터
정답: d. 정형 데이터 / 반정형 데이터
스키마를 가진 데이터는 정형 데이터, $\text{Tag}$나 $\text{Marker}$로 유연한 구조를 가진 데이터는 반정형 데이터입니다.
7. 두 변수 간의 관계를 산점도로 나타냈을 때, 데이터 포인트들이 우상향하는 직선 주변에 매우 가깝게 밀집되어 있다면 피어슨 상관계수($r$) 값은 대략 얼마에 가까울까요?
  • a. $r \approx 0.0$
  • b. $r \approx -0.9$
  • c. $r \approx 0.9$
  • d. $r \approx -0.5$
정답: c. $r \approx 0.9$
우상향하는 관계는 양의 상관관계(1에 가까움)이며, 밀집도가 높으면 선형성이 강하므로 $r \approx 0.9$와 같이 높은 양의 값을 가집니다.
8. 특정 제품의 시장 점유율 평균이 기존에 알려진 기준값($50\%$)과 통계적으로 유의미하게 차이가 나는지 검증하고자 합니다. 이때 가장 적절한 가설 검증 방법은 무엇인가요?
  • a. 단일 표본 t-test
  • b. 대응 표본 t-test
  • c. 독립 표본 t-test
  • d. One-way ANOVA
정답: a. 단일 표본 t-test
단일 표본 t-test는 한 변수의 평균값이 **알려진 특정 평균값**과 일치하는지 검정할 때 사용됩니다.
9. 데이터 분석의 상세 과정 8단계 중, **데이터 전처리** 이후에 수행되며 **모델링** 이전에 수행되는 단계는 무엇인가요?
  • a. 데이터 수집 (Data Collection)
  • b. 탐색적 데이터 분석 (EDA)
  • c. 분석 및 평가 (Analysis & Estimation)
  • d. 해석 (Interpretation)
정답: b. 탐색적 데이터 분석 (EDA)
상세 과정 순서는 ... $\rightarrow$ 데이터 전처리 $\rightarrow$ **탐색적 분석** $\rightarrow$ 모델링 $\rightarrow$ ...입니다.
10. 선형 회귀($y=Ax+B$) 모델을 학습시키는 과정에서 **회귀 모형과 실제 데이터 간의 오차**를 최소화하도록 하는 것이 핵심 원리입니다. 이 오차를 나타내는 용어는 무엇인가요?
  • a. 잔차 (Residual) 또는 오차 (Error)
  • b. 피어슨 $r$ 값
  • c. 공분산
  • d. 표준편차
정답: a. 잔차 (Residual) 또는 오차 (Error)
회귀에서 실제 관측값과 회귀선 상의 예측값 사이의 차이를 잔차 또는 오차라고 하며, 회귀 분석은 이 잔차의 합을 최소화하는 것을 목표로 합니다.
